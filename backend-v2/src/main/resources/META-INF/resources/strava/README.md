# Strava brand assets

These SVGs (`btn_strava_connect.svg`, `powered_by_strava.svg`) are **faithful,
brand-spec recreations** in Strava orange (`#FC4C02`) used for the "Connect with
Strava" button and "Powered by Strava" attribution.

**Before public launch, replace them with the EXACT official artwork** from Strava's
brand kit, linked from <https://developers.strava.com/guidelines/> (downloadable
EPS/SVG/PNG). Strava's guidelines require their provided artwork, unmodified:
- Connect with Strava button: `btn_strava_connect_with_orange` (≥48px tall)
- Powered by Strava logo: `api_logo_pwrdBy_strava_horiz_orange` (+ white/black variants)

Do not modify, recolor, or animate the official marks; never use them as the app icon.
Keep the file names referenced by `LoginScreen.tsx` / `Setup.tsx` / `App.tsx`, or
update those references when swapping in the official files.
